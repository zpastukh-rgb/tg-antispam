# Скриншоты для подсказок бота

Положите сюда изображения для пошаговой подсказки «Добавить бота в группу»:

- **addgroup_step1.png** — скриншот: где нажать кнопку под полем ввода (шаг 1).
- **addgroup_step2.png** — скриншот: выбор группы и выдача прав (шаг 2).

Формат: PNG, JPG или WEBP. Если файлов нет, бот отправит только текст и кнопку.
